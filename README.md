# dw_handshake

dw_handshake is a lightweight FiveM script that allows players to perform
a simple handshake with the nearest player using a keybind.

The script is fully client-side, has no dependencies and is designed to improve
social roleplay interactions in a clean and minimal way.

---

## Features
- Handshake with nearest player
- Triggered via keybind (default: P)
- No UI
- No framework required
- Minimal performance impact

---

## Requirements
- FiveM client

---

## Installation
1. Download or clone this repository
2. Place the folder `dw_handshake` into your `resources` directory
3. Add the following line to your `server.cfg`: ensure dw_handshake
---

## Notes
- Players must be close to each other
- Only the local player animation is played (RP-based interaction)
- Keybind can be changed in FiveM settings
